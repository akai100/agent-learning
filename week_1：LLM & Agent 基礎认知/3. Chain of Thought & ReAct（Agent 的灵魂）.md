**ReAct 的真正价值**

ReAct =
**Reason + Act 的循环**

不是为了好看，而是为了：

+ 插入工具

+ 插入判断

+ 插入纠错

**你要理解这个循环**

```makefile
Thought: 我现在需要什么？
Action: 调用工具
Observation: 得到了什么？
Thought: 下一步做什么？
```

👉 这就是 Agent 的“CPU”
