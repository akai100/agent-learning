
从今天开始，你要把 Prompt 当成一门 DSL（领域语言）

**Prompt 的真实结构（非常重要）**

```sql
System Prompt   → 决定“人格和边界”
User Prompt     → 提供任务
Hidden Context  → 记忆、工具说明
```

**❗ 关键认知**

System Prompt 是“宪法”

如果你后面发现 Agent 不听话，90% 是 System Prompt 设计失败。

**你必须掌握的 3 类 Prompt**

**1️⃣ 角色 Prompt**

不是“你是一个专家”，而是：

+ 你负责什么

+ 你不负责什么

+ 你失败时怎么办

**2️⃣ 过程 Prompt（为 Agent 服务）**

例子（思想，不是模板）：

在给出最终答案前，你必须先输出你的推理步骤

👉 这是 Reasoning 的开关

**3️⃣ 约束 Prompt（防胡说）**

+ 不知道就说不知道

+ 必须引用来源

+ 输出格式固定

**实战**

🎯 目标

写 3 个 Prompt，解决同一个问题：

1. 普通问法

2. 带角色 + 过程

3. 带约束

对比输出质量。
